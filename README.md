# Wayangku 
Wayangku adalah aplikasi yang berisi informasi tentang wayang, yaitu : 
1. Profile Wayang 
2. Jenis Wayang
3. Cerita Wayang
4. Video Wayang
Tujuan kami membuat aplikasi ini untuk mempermudah kita dalam mencari info seputar wayang di Indonesia.
